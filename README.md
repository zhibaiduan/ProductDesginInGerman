# Germany Workplace Compliance Handbook

Static handbook for people preparing to work in German or European teams.

## Deploy

This repo is ready for Vercel as a static site. Use the repository root as the project root.

Main files:

- `index.html` - handbook entry page
- `styles.css` - shared styling
- `app.js` - navigation, quiz, translations, and feedback submission
- `pretest.html` - embedded beginner score check
- `vercel.json` - clean URLs and legacy redirects

No build command is required.
